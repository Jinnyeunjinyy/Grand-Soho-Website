# Grand Soho 웹사이트 기획서

**작성일**: 2026-06-14  
**프로젝트**: Grand Soho 공식 웹사이트  
**운영사**: 그랜드벤처스(GrandVentures)

---

## 1. 프로젝트 개요

### 배경

Grand Soho는 VC 그랜드벤처스가 직접 운영하는 강남 소재 스타트업 공유오피스입니다. 단순 공간 임대를 넘어, 입주사에게 투자·멘토링·법률 자문·오픈이노베이션 등 스타트업 성장 파트너 프로그램을 함께 제공하는 것이 핵심 차별점입니다.

### 목표

- 잠재 입주 고객이 Grand Soho의 공간 및 파트너 프로그램 가치를 직관적으로 이해하도록 한다
- 입주 상담·공간 예약으로의 전환(Conversion)을 유도한다
- 기존 입주 고객이 공지사항·예약 정보를 편리하게 확인할 수 있도록 한다

---

## 2. 사용자 정의 (User Personas)

| 유형 | 설명 |
|------|------|
| **잠재 입주 고객** | 시드~Series A 직전·직후 스타트업 창업자·공동창업자. 투자 유치 전후 사무공간과 성장 지원을 함께 찾는 5~10인 팀 |
| **기존 입주 고객** | 공간 예약 및 파트너 프로그램을 이용하는 현재 입주사 멤버 |
| **운영자** | 공간 관리, 입주 문의 처리, 공지사항 관리를 담당하는 관리자 |

---

## 3. 핵심 셀링 포인트

Grand Soho는 VC 그랜드벤처스가 직접 운영하는 강남의 스타트업 공간입니다. 다른 공유오피스와 달리, 입주사에게는 스타트업 파트너 프로그램이 제공됩니다.

### 파트너 프로그램 4대 혜택

| 혜택 | 내용 |
|------|------|
| **오픈이노베이션 · PoC** | (주)오메가포인트와의 에듀테크·콘텐츠 분야 사업성 검증 기회 (25년 브랜드 시매쓰 운영사) |
| **투자·성장 멘토링** | 그랜드벤처스 파트너 직접 참여, 벤처투자·성장 전략 컨설팅 월 1시간 |
| **법률·컴플라이언스 자문** | 변호사 자격 보유 이하연 파트너, 스타트업 리스크·법률 자문 월 30분 |
| **포트폴리오 네트워킹** | 그랜드벤처스 포트폴리오사·파트너사와의 네트워킹 기회 |

## 4. 페이지 구조 (Information Architecture)

```
/ ──────────────── 홈 (Hero → Features → Partner Program → Gallery → Location)
/spaces ─────────── 공간 안내 · 요금
/contact ────────── 문의 폼
/notice ─────────── 공지사항 목록
/notice/:id ─────── 공지사항 상세
[외부 ↗] ─────────── 네이버 예약 페이지 (새 탭)
```

---

## 5. 페이지별 기능 정의 (MVP)

### 5-1. 홈 (`/`)

**헤드라인 카피**: "공간도 전략입니다."  
**서브카피**: 그랜드소호는 투자사 그랜드벤처스가 함께하는 스타트업 거점입니다. 멘토링, 법률 자문, 오픈이노베이션 — 입주사라면 누구나 시작할 수 있습니다.

#### 홈 섹션 구조

| 순서 | 섹션 | 목적 |
|------|------|------|
| 1 | **Hero** | 헤드라인 + 서브카피 + CTA 2종 (입주 문의, 예약하기) |
| 2 | **Feature** | 공간 편의 4대 특징 카드 |
| 3 | **Partner Program** ★신설 | VC 운영 파트너 프로그램 4대 혜택 카드 |
| 4 | **Gallery** | 공간 사진 갤러리 + Lightbox |
| 5 | **Location** | 카카오맵 + 주소·교통·연락처 |

#### Feature 카드 4종

- 유연한 계약
- 올인원 관리
- 강남 위치
- 감성 공간

---

### 5-2. 공간 안내 (`/spaces`)

- 공간 유형별 탭: 개인실 · 회의실 · 라운지
- 요금 카드 (2-3인실 · 4인실 · 미팅룸 · 라운지)
- 네이버 예약 외부 링크 CTA

---

### 5-3. 예약 (외부 링크)

- "예약하기 ↗" 버튼 클릭 시 네이버 예약 페이지 새 탭으로 외부 이동
- 별도 예약 페이지 없음
- 모든 페이지에서 GNB·CTA로 접근 가능

---

### 5-4. 문의 (`/contact`)

#### 문의 폼 필드

| 필드 | 타입 | 필수 |
|------|------|------|
| 이름 | Text Input | Required |
| 연락처 | Text Input | Required |
| 문의 유형 | Select (입주 상담 / 회의실 / 기타) | Required |
| 문의 내용 | Textarea | Required |

#### 기술 사항

- **이메일 전송**: EmailJS 사용, 운영자 이메일(`ehong@gvc.im`)로 전송
- **제출 결과 처리**: 성공·실패 인라인 메시지 (별도 페이지 이동 없음)

---

### 5-5. 공지사항 (`/notice`, `/notice/:id`)

- 운영시간 변경, 이벤트, 공실 안내 등 간단한 공지
- 목록 페이지: 제목 · 날짜 · 카테고리 표시
- 상세 페이지: 본문 내용 + 이전/다음 글 이동

---

## 6. GNB (Global Navigation Bar)

| 메뉴 | 링크 |
|------|------|
| Grand Soho (로고) | `/` |
| 공간 안내 | `/spaces` |
| 공지사항 | `/notice` |
| 문의 | `/contact` |
| 예약하기 ↗ | 네이버 예약 (외부, 새 탭) |

- 모바일: 햄버거 메뉴로 전환
- 스크롤 시 배경 변화 (투명 → 불투명)
